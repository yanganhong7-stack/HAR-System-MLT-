# Human Activity Recognition with Wearable Sensors

This repository contains a machine learning pipeline for **Human Activity Recognition (HAR)** using wearable sensor data from back and thigh accelerometers.

The project was developed for an MLT coursework task and compares three modelling approaches:

1. **Semi-supervised K-Means** with handcrafted statistical features
2. **Random Forest** with baseline and fine-tuned feature sets
3. **1D CNN** using raw sliding-window signals plus ENMO channels

## Project Highlights

- Cleans wearable sensor time-series data and handles the anomalous `S007` recording.
- Removes cycling labels and merges ascending/descending stairs into one unified class.
- Converts continuous sensor readings into 2-second sliding windows with 1-second overlap.
- Extracts ENMO-based and axis-level statistical features.
- Evaluates models with 5-fold cross-validation and final test-set performance.
- Produces classification reports, confusion matrices, and model comparison tables.

## Main Methods

### Sliding Window Generation

Continuous time-series data is segmented using:

- Window length: **2 seconds**
- Overlap: **1 second**
- Default sampling rate: **100 Hz**

Each window receives the majority label inside that window.

### Baseline Features

The baseline feature set contains 10 features:

- ENMO mean and standard deviation for back and thigh sensors
- Mean and standard deviation for selected raw axes

### Fine-Tuned Features

The fine-tuned feature set contains 20 features:

- ENMO mean, standard deviation, maximum, and energy
- Mean and standard deviation for all 6 acceleration axes

### Models

| Model | Description |
|---|---|
| K-Means | Semi-supervised mapping from clusters to majority training labels |
| Random Forest | Baseline and class-balanced fine-tuned configurations |
| 1D CNN | Baseline shallow CNN and fine-tuned CNN with BatchNorm, Dropout, and GlobalAveragePooling |

## Reported Coursework Result

In the coursework experiment, the fine-tuned Random Forest achieved approximately:

- **Accuracy:** 91.76%
- **Macro Recall:** 82.03%

Rerun results may vary depending on dataset version, random seed, TensorFlow version, and hardware.
